# Pixel Studio - Simple Photo Editor
A simple photo editor built with Next.js and Tailwind CSS.

## Features
- Crop images
- Apply filters
- Adjust brightness, contrast, and saturation

## How to use
1. Clone the repository with `https://github.com/lightdarkmaster/Photo-Editor`
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Open the photo editor at `http://localhost:3000`
